# Assets

High-resolution versions of all images, diagrams, charts, and flowcharts
used in the book.

## Naming Convention

`fig-XX-YY-description.png`
- `XX` = chapter number
- `YY` = figure number within chapter
- `description` = brief label

## Formats

| Format | Use Case |
|--------|----------|
| `.png` | Diagrams, screenshots, infographics |
| `.svg` | Scalable vector graphics (preferred for diagrams) |
| `.pdf` | Print-quality figures |
